Busamed Event and Clinic Management
Project Overview

This project is a comprehensive Healthcare Management Platform developed for Busamed Healthcare, a South African medical provider. It serves as an enterprise wellness event and patient care system, enabling companies to book medical screenings, nurses to conduct on-site consultations, and administrators to manage events and patient data in real time. The platform currently operates across multiple Busamed clinics, supporting concurrent wellness events and individual consultations.

Technologies Used

Frontend: React 18 Vite Tailwind CSS DaisyUI
Backend: Node.js Express.js MongoDB JWT Auth
AI & Cloud: OpenAI API Google Cloud Vercel

Key Features
  • Multi-User Role System: Separate access for companies, nurses, admins, patients, and super admins
  • Wellness Event Management: Corporate booking, scheduling, and multi-site coordination
  • Medical Consultation System: SOAP note documentation, vitals collection, and AI-assisted diagnosis
  • Patient Management: Medical history, registration, referrals, and billing integration
  • Analytics Dashboard: Real-time health data insights, event statistics, and PDF reporting
Impact & Highlights
  • Handles simultaneous wellness events across 3 clinics
  • Supports 5 distinct user roles with secure access control
  • Automates health report generation and documentation
  • Provides AI-powered consultation assistance and analytics
